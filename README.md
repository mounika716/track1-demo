AI Influencer Intelligence Engine 🤖📊
An advanced web-based platform designed to analyze influencer profiles using simulated Machine Learning models. This tool moves beyond vanity metrics (like follower counts) to provide actionable insights on authenticity, growth potential, and brand compatibility.

🌟 Features
Influencer Data Collection: Input key metrics such as followers, engagement rates, and posting frequency.
Authenticity Detection Engine: Analyzes engagement patterns to detect potential bot activity or purchased followers.
Growth Prediction: Uses a heuristic model to predict future audience expansion based on content category and consistency.
Brand Matching System (RAG Simulation): Uses keyword overlap and NLP concepts to match creators with brand descriptions.
Ratefluencer Score™: A composite 0-100 score ranking creators by predicted business impact.
🚀 Getting Started
This is a static web application that runs entirely in the browser. No backend installation is required.

Prerequisites
A modern web browser (Chrome, Firefox, Edge, Safari).
A code editor (VS Code recommended).
Installation
Clone the repository:
git clone https://github.com/your-username/track1-demo.gitcd track1-demo

📖 Usage Guide
Input Data: Enter an influencer's handle, niche, and statistics (Followers, Likes, Comments).
Brand Context: Enter a target brand description (e.g., "Nike, Gymshark") to test the matching algorithm.
Analyze: Click "Run Intelligence Engine".
View Results: The dashboard will display:
Ratefluencer Score: The overall value rating.
Sub-scores: Breakdown of Authenticity, Growth, Brand Match, and Engagement Quality.
AI Insights: Text-based explanations of the scores.
Quick Demo
Don't have data ready? Click the "Load Demo Data" button to populate the form with a realistic Tech Influencer profile.

🛠️ Tech Stack
HTML5: Semantic structure.
CSS3: Flexbox/Grid layouts, CSS Variables for theming, Animations.
Vanilla JavaScript: Logic for scoring algorithms, DOM manipulation, and simulated ML processing. No external frameworks.
